---
title: 'Объединение данных'
---

Сервис сбора данных StreamMyData позволяет передавать различные данные в удобный для вас СУБД.
После того, как вы передадите все необходимые данные в СУБД вы можете приступить к процедуре объединения множества переданных наборов данных в единую структуру.
Объединение набора данных внутри СУБД осуществляется при помощи “ключей”. “Ключ” — уникальный идентификатор, который должен находиться в двух разных таблицах для их объединения.
Например, идентификатор рекламной компании. Вы можете обнаружить его в выгрузке данных из рекламной системы Яндекс.Директ. Этот же идентификатор можно извлечь из utm-метки, которую зафиксировала веб-аналитическая система.
Обнаружив эти два идентификатора в двух таблицах у специалиста появляется возможность соединить статистические данные из выгрузки Яндекс.Директ с данными о том, как пользователь вел себя на сайте из веб-аналитической системы.
Далее вы можете по идентификатору транзакции сопоставить данные о входящей выручке в веб-аналитической системе с данными по тому же идентификатору о реализации в вашей CRM.

Таким образом специалист, зная “Ключи” в наборах данных, может сопоставлять множество разных выгрузок и анализировать их.

После того, как связи между различными выгрузками (наборами данных) созданы у специалиста появляется возможность создавать запросы к данным для того, чтобы смешивать любые параметры и метрики из всех этих данных и получать удивительные инсайты для бизнеса.
